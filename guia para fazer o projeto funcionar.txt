GUIA OFICIAL DE INSTALAÇÃO – ASCENDE DASHBOARD
(versão atualizada e organizada)

Verificar versões instaladas
Execute os comandos abaixo para verificar se tudo está instalado corretamente:

python --version
pip --version
git --version




Requisitos: Python 3.11 ou 3.12 (não usar Python 3.13+).

Criar e ativar o ambiente virtual
Dentro da pasta onde o projeto ficará salvo, execute:

py -3.12 -m venv ascende_env

Ative o ambiente:

ascende_env\Scripts\activate






Atualizar o pip

python -m pip install --upgrade pip

Instalar dependências essenciais

pip install streamlit pandas duckdb plotly matplotlib wordcloud firebase-admin requests numpy

Observação:

PyArrow é opcional.

Caso pyarrow não funcione, ignore. O DuckDB já faz o trabalho de leitura de arquivos Parquet/Arrow.

Se desejar tentar instalar:

pip install pyarrow







Configurar o Firebase

No código (entre as linhas 180 e 184), existe:

cred_path_firebase = r"C:\Users\thiag\Documents\ASCENDE-DASHBOARD\ascende-firebase.json"
df_vagas = carregar_dados_firebase(cred_path_firebase)
if df_vagas.empty:
st.warning("Nenhuma vaga encontrada.")
st.stop()

O usuário deve substituir duas coisas:

O caminho do arquivo .json do Firebase (cred_path_firebase).
Exemplo:
cred_path_firebase = r"C:\Users\SEU_USUARIO\Documents\ASCENDE-DASHBOARD\ascende-firebase.json"

O arquivo JSON do Firebase, que deve ser fornecido pelo proprietário do projeto.

Sem esse arquivo, o Streamlit não conseguirá acessar o banco de dados.








Executar o projeto

streamlit run ascende_dashboard_template.py

O navegador abrirá automaticamente o dashboard.
